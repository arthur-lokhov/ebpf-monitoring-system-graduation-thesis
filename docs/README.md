# eBPF Monitor

Платформа мониторинга на основе eBPF с поддержкой плагинов и интеграцией с Prometheus.

## Описание

**eBPF Monitor** — это современная система мониторинга, которая использует технологию eBPF для сбора метрик в реальном времени с минимальными накладными расходами.

## Ключевые возможности

- 🧩 **Плагины** — архитектура с поддержкой плагинов (аналогично дашбордам в Grafana)
- 📊 **Prometheus** — встроенный экспортер метрик для интеграции с Prometheus
- 📦 **Экосистема плагинов** — централизованный реестри для загрузки и обмена плагинами

## Технологии

| Компонент | Технология |
|-----------|------------|
| Backend | Go |
| Frontend | Svelte |
| eBPF | libbpf / bpftrace |
| Метрики | Prometheus |

## Структура проекта

```
ebpf-monitoring/
├── cmd/           # Точки входа приложений
├── internal/      # Внутренние пакеты
├── plugins/       # Система плагинов
├── pkg/           # Публичные пакеты
├── web/           # Frontend на Svelte
├── docs/          # Документация
└── configs/       # Конфигурационные файлы
```
